I work on LLM engineering, AI agents, Retrieval-Augmented Generation, prompt engineering, model evaluation, function calling, multimodal AI, fine-tuning, local LLMs, and production-ready AI interfaces.

I completed the AI Engineer Core Track on LLM engineering, RAG, QLoRA, and agents. The course was project-heavy, which suits me. I learn best by building the thing, breaking it, fixing it, and turning it into a product surface someone can try.

Before moving into LLM engineering, I built products with React, Next.js, TypeScript, React Native, Node.js, Python, GraphQL, Supabase, and PostgreSQL. That helps when an AI prototype needs to become a usable product instead of staying inside a notebook.
